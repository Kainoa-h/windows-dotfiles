Rough windows dotfiles.

Clone into user home.

- GlazeWM & Zebar for tiling window management and bar.
    - symlink both dirs into `%USERPROFILE%/.glzr/`
- Alacritty for terminal
    - symlink alacritty.toml to `%APPDATA%/alacritty/alacritty.toml`
- Kanata for keyboard hehe
    - download release and place winIO GUI in user home
    - run kan.vbs to start
- Install nixos-wsl and load wsl-host profile from nix repo
